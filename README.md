### bookstore
> 采用Vue、Koa、Zepto、localStorage、Base64、Jsonp、SCSS、Gulp开发一个面向移动端的书城整站。项目中遇到几个比较复杂的模块，如阅读器，书城首页，搜索模块等。实现整站的主页、书籍详情页、书籍列表页、个人主页、搜索功能。采用Vue数据驱动，Koa搭建服务器运行环境，数据Mock


**技术栈**

- vue
- ES6
- Koa
- Zepto
- localStorage
- Jsonp
- SCSS
- Gulp


Usage
---

- 安装依赖 

```
npm install
```

- 切换到项目根目录执行

```
node app.js
```
